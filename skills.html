  • An E-Commerce Furniture Store

    A spring boot application that enables a customer to signup, login and purchase furniture online. The application has two modules - admin and customer. The Admin can add and delete categories and furniture items to the application for the customers to purchase them. The Customer can purchase the furniture items that are added by the user. The application uses Spring Boot, Spring Web, Spring Security, Spring MVC, Spring Hiberanate, Thymeleaf and MySQL. All the passwords stored in the system are encrypted using bcrypt algorithm.

  • Serve Clever

    An application built out of HTML, CSS and JavaScript that enables hotels and waiters to manage the orders made by the customers. This application uses an API to fetch that data of the food items in the menu. The food items can be searched in the menu based on the name and its category. The tables can also be searched based on the table number. The food items can be added to a particular table using a drag and drop feature. Once the item is dropped, the total cost of the table is also updated. The App uses session storage to store the table information.
